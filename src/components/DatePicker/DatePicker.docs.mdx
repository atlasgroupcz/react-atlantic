# DatePicker

-   This component is dependent on https://github.com/Hacker0x01/react-datepicker/ with custom input and header components
-   Dependency docs https://reactdatepicker.com/

## Common usage

<Playground>
    {() => {
        const [selectedDate, setSelectedDate] = React.useState(new Date());
        return (
            <div style={{ width: 200 }}>
                <DatePicker
                    todayButton={`Dnes`}
                    placeholderText={`Vyberte datum`}
                    isFullWidth
                    dateFormat="dd. MM. yyyy"
                    selected={selectedDate}
                    onChange={setSelectedDate}
                    minDate={new Date('2021/01/01')}
                />
            </div>
        );
    }}
</Playground>
