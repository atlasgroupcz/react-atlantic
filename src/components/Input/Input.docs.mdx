## Hooks variants

It's highly reccomended to use custom hooks instead default ones provided by this lib, for more info see [know bugs section](#known-bugs)

```tsx
import { InputDefaultView, useInputChange, useInputClearable } from '.';
import { wrap } from '@atlasgroup/react-wrap';

export const Inputs = {
    Default: wrap(InputDefaultView, useInputChange),
    Clearable: wrap(InputDefaultView, useInputClearable),
};
```

## Known bugs

-   `useInputClearable` clears value on input click
-   hooks do not accept nor pass props that are not handled by them
-   `useInputFloatingLabel` wraps `useInputDefault` nothing else
-   `InputFloatingLabelView` label overlflows input value

<Playground>
    <Inputs.Clearable defaultValue="Value to be cleared" />
    <InputFloatingLabelView value="Value to be cleared" label={'text which overflows value'} />
</Playground>
