# Switch

Is a component that allows the user to toggle between two states.

## Basic usage

<Playground>
    <WrapperVertical>
        <Switch />
        <Switch isChecked />
    </WrapperVertical>
</Playground>

## Using different types

<Playground>
    <WrapperVertical>
        <Switch type="default" isChecked />
        <Switch type="primary" isChecked />
        <Switch type="info" isChecked />
        <Switch type="success" isChecked />
        <Switch type="warning" isChecked />
        <Switch type="error" isChecked />
    </WrapperVertical>
</Playground>

## Using on & off texts

<Playground>
    <WrapperVertical>
        <Switch textOn="On" isChecked />
        <Switch textOff="Off" isChecked={false} />
    </WrapperVertical>
</Playground>

## Using check states

<Playground>
    <WrapperVertical>
        <Switch isChecked={false} isPartiallyChecked />
        <Switch isChecked isPartiallyChecked />
        <Switch isChecked={false} isPartiallyChecked={false} />
    </WrapperVertical>
</Playground>

## Using disabled state

<Playground>
    <WrapperVertical>
        <Switch isChecked isDisabled />
